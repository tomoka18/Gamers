# Gamers

## サイト概要
- 気軽にゲームのフレンドを募集したり、募集してる人を探せるサイトです

### サイトテーマ
- ゲームのフレンド募集サイト

### テーマを選んだ理由
- ゲームが趣味で気軽にフレンドを探せたらいいなと思ったため

### ターゲットユーザ
- 一緒にゲームをやる人を探している人

### 主な利用シーン
- 誰かと一緒にゲームがしたいなと思った時
